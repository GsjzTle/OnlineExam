# OnlineExam

## 在线考试系统

### 安装步骤


#### 1. 克隆项目到本地

```git
git clone https://gitee.com/gsjztle/online-exam.git

```



#### 2. 进入前端文件夹并安装模块

```git
cd Frontend
npm install 
```


#### 3. 运行前端项目 

```git

npm run dev
```

> 前端端口默认为 3030



#### 4. 用 idea 打开 Backend，并加载 pom.xml


#### 5. 待依赖加载完后自动项也会自动配置好，点击编辑器上方的绿色运行按钮即可

> 后端端口默认为 8008
